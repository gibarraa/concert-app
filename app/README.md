ConcertApp
Bienvenido a ConcertApp, una aplicación moderna de Android desarrollada con Jetpack Compose para descubrir y comprar entradas para tus conciertos favoritos.
Características Principales
Este proyecto demuestra el uso de UI moderna y animaciones fluidas en Android:
Exploración de Eventos: Visualiza conciertos destacados y próximos eventos en una interfaz inmersiva.
Detalle de Concierto: Información detallada con imágenes de alta calidad, fechas y ubicación.
Transiciones Animadas: Navegación fluida entre pantallas con efectos de deslizamiento (slide-in/out).
Micro-interacciones:
Animaciones de entrada en las tarjetas de eventos (Fade & Slide Up).
Efecto de "rebote" al marcar un evento como Favorito.
Pruebas de UI: Tests automatizados para verificar la navegación crítica.
Tecnologías Utilizadas
Lenguaje: Kotlin
UI Toolkit: Jetpack Compose
Navegación: Navigation Compose + Accompanist (para animaciones).
Carga de Imágenes: Coil
Inyección de Dependencias: Hilt (si aplica) o ViewModel manual.
Pruebas: JUnit4 + Compose UI Test.
Instalación y Ejecución
Clonar el repositorio: https://github.com/gibarraa/concert-app.git
Abrir en Android Studio:
Selecciona la carpeta raíz del proyecto.
Sincronizar Gradle:
Espera a que se descarguen las dependencias.
Ejecutar:
Conecta un dispositivo o inicia un emulador y presiona Run
Ejecutar Pruebas
Para verificar que la navegación funciona correctamente:
Ve a la carpeta androidTest.
Haz clic derecho en NavigationTest.kt.
Selecciona Run 'NavigationTest'.
